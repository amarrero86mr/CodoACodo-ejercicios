Práctica

  • Crear una función que haga lo siguiente: crear un array vacío, que se irá llenando de una lista de productos, deberemos usar un bucle para que nos vaya preguntando qué producto agregar. Sólo dejará de hacer el bucle cuando le digamos que no queremos seguir agregando productos.
  • Cada producto se ingresará en un array vacío, la función terminará al retornar un ARRAY DE OBJETOS.
  • Recorreremos el array de objetos y usaremos métodos aprendidos.
